# Finnish Online Forums

Tools and datasets for collecting, processing, and analysing data from Finnish online discussion forums.

## Overview

Finnish online forums — including platforms like Suomi24, Ylilauta, and others — are a rich source of naturalistic text data for social research. This project provides:

- Data collection scripts
- Preprocessing pipelines
- Documentation on data structure and ethical use
